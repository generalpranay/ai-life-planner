# AI Life Planner

AI Life Planner is a comprehensive application designed to help users organize their lives with the help of AI. It consists of a Node.js/Express backend and a Flutter mobile application.

## Project Structure

- `backend/`: Node.js/Express API server.
- `mobile_app/`: Flutter mobile application.

## Getting Started

### Backend

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Mobile App

1. Navigate to the mobile app directory:
   ```bash
   cd mobile_app
   ```
2. Get dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app:
   ```bash
   flutter run
   ```

## Testing

To run backend tests:

```bash
npm run test:backend
```
